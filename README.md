# DPlus - Premium Video Downloader

DPlus is a simple video downloader for Windows. It provides a seamless experience for downloading videos from major platforms including YouTube, Twitter (X), Facebook, and Instagram.

![DPlus Preview]()

## 🚀 Features

- **Multi-Platform Support**: Download from YouTube, Twitter, Facebook, and Instagram.
- **Intelligent Naming**: Automatically fetches the original video title for filename suggestions.
- **Real-time Progress**: Precise 1% increment progress tracking.
- **Premium UI**: Modern glassmorphic design with smooth micro-animations.
- **Portable**: Single `.exe` file—no installation required.
- **Auto-Organization**: Downloads are saved to a dedicated `DPlus` folder in your Downloads directory and opened automatically.

## 📦 Downloads

The latest **Portable Windows version** is available in the [GitHub Releases](https://github.com/Kornilio/DPlus/releases) section. 

Just download the `DPlus_Portable_x.x.x.exe` and you're ready to go!

## 🛠️ Performance & Tech Stack

- **Framework**: Electron + React + Vite
- **Engine**: yt-dlp (Bundled & Optimized)
- **Styling**: Vanilla CSS (Glassmorphism)
- **Language**: TypeScript

## 📝 Usage

1. Paste your video link into the input field.
2. Click **Download**.
3. (Optional) Customize the filename in the auto-focused modal.
